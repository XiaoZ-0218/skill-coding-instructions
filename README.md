# coding-instructions

一套面向 AI 辅助编程的极简主义工作流规范：

- **Andrej Karpathy 编程准则**：从简开始、先复制再创新、理解底层、循序渐进、最小依赖。
- **Gitmoji 提交与分支工作流**：功能分支开发、频繁提交、测试通过后合并、清晰的提交历史。

适用于需要严格遵循极简编码风格、频繁提交、功能分支开发与测试通过后再合并的项目。

## 使用方式

在 Kimi Code 等支持 skill 的 AI 编程助手中引用本规范，让 AI 代理在编码与提交过程中遵循统一的工作流。
